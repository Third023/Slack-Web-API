# Slack Web API Project

## Overview
This project is a web application that interacts with the **Slack API** to send messages, manage channels, and retrieve user information. It is built using **HTML, CSS, JavaScript (Node.js/Express)** and demonstrates API integration with OAuth 2.0 authentication.

---

## Features
- **Send Messages** to a Slack channel
- **Retrieve Channel List** from your Slack workspace
- **Retrieve Users List** from your workspace
- **Authentication** using OAuth 2.0 / Bot Token
- **Error Handling** for failed API requests
- **Frontend & Backend Separation** for modularity

---

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Third023/Slack-Web-API.git

```
2. Navigate to the project folder:

cd Slack-Web-API

3. Install dependencies (Node.js required):

npm install

4. Create a .env file in the root directory with your Slack credentials:

CLIENT_ID=your_slack_client_id
CLIENT_SECRET=your_slack_client_secret
BOT_TOKEN=your_slack_bot_token
PORT=3000

5. Start the backend server:

node server.js
